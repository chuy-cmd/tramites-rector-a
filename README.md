## Subir a GitHub Pages

```bash
git init
git add .
git commit -m "Sitio expo Trámites de Rectoría UTC"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/TU_REPO.git
git push -u origin main
```

Luego: GitHub → Settings → Pages → Source: `main / root` → Save.
En ~2 minutos tendrás la URL pública.
